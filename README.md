# MiWallet - Billetera Virtual

Trabajo Practico Final - Programacion Orientada a Objetos en Python

Simulador de billetera virtual con interfaz grafica desarrollado en Python usando Tkinter. Permite gestionar usuarios y realizar operaciones financieras basicas como depositos, transferencias y pagos con QR.

---

## Funcionalidades

- Depositar dinero en una billetera
- Transferir dinero entre usuarios
- Pagar con QR en comercios
- Ver historial de transacciones
- Crear, modificar y eliminar usuarios

---

## Clases utilizadas

| Clase | Descripcion |
|---|---|
| `Usuario` | Representa a cada persona con su nombre, email y DNI |
| `Billetera` | Maneja el saldo y el historial de cada usuario |
| `Transaccion` | Registra cada operacion con su tipo, monto y hora |

---

## Usuarios de ejemplo

El sistema carga automaticamente dos usuarios al iniciar:

- Maria Garcia - saldo inicial $25.000
- Carlos Lopez - saldo inicial $5.000
